## Как протестировать

### Тест 1: Консольный вывод (текущая настройка)

1. Убедитесь что в `.env` установлено:
   ```
   EMAIL_BACKEND=django.core.mail.backends.console.EmailBackend
   ```

2. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

3. Добавьте товары в корзину

4. Нажмите "Оформить заказ"

5. Заполните форму и отправьте

6. **Смотрите в консоль (терминал)** - там появится содержимое письма

### Тест 2: Реальная отправка Email

1. Обновите `.env` (см. EMAIL_SETUP.md для подробностей):
   ```env
   EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
   EMAIL_HOST=smtp.gmail.com
   EMAIL_HOST_USER=ваш-email@gmail.com
   EMAIL_HOST_PASSWORD=ваш-app-password
   DEFAULT_FROM_EMAIL=ваш-email@gmail.com
   ```

2. Перезапустите сервер:
   ```bash
   python manage.py runserver
   ```

3. Оформите заказ как в Тесте 1

4. Проверьте почту **store@cascate.ru** - должно прийти письмо с PDF

## Возможные проблемы

### Ошибка при отправке email
- Проверьте настройки SMTP в `.env`
- Для Gmail убедитесь что используете App Password, а не обычный пароль
- Проверьте что двухфакторная аутентификация включена

### Изображения не появляются в PDF
- Убедитесь что товары имеют загруженные изображения
- Проверьте что путь к media файлам корректен

### Логотип не появляется в PDF
- Убедитесь что файл `static/images/logo-navbar.png` существует
- Запустите `python manage.py collectstatic` если нужно
