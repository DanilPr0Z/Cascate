# Cascate Porte - Интернет-магазин дизайнерской мебели

Django-приложение для каталога итальянской мебели Cascate Porte e mobile.

## Описание

Полнофункциональный интернет-магазин с каталогом мебели, корзиной покупок, системой навигации по категориям и современным дизайном.

## Возможности

- 🛋️ **Каталог товаров** с категориями и подкатегориями
- 🛒 **Корзина покупок** с расчетом итоговой суммы
- 📱 **Адаптивный дизайн** для всех устройств
- 🎨 **Современный UI** с плавными анимациями
- 🔍 **Сортировка товаров** по различным параметрам
- 🖼️ **Галерея изображений** для каждого товара
- ⭐ **Система рейтингов** товаров
- 📊 **QR-коды** для каждого товара
- 🏪 **Управление складами** и остатками

## Технологии

- **Backend**: Django 5.0
- **Frontend**: HTML5, CSS3, JavaScript (без системы авторизации)
- **База данных**: SQLite (для разработки) / PostgreSQL (рекомендуется для продакшена)
- **Шрифты**: PT Sans
- **Цветовая схема**: Серые оттенки (#444, #333, #666)

## Установка и запуск

### 1. Клонирование и установка зависимостей

```bash
# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка базы данных

```bash
# Примените миграции
python manage.py migrate

# Создайте суперпользователя для админ-панели
python manage.py createsuperuser
```

### 3. Запуск сервера разработки

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: `http://127.0.0.1:8000/`
Админ-панель: `http://127.0.0.1:8000/admin/`

## Модели базы данных

### Основные модели:

- **Category** - Главные категории (Мягкая мебель, Двери и перегородки, Системы хранения и т.д.)
- **SubCategory** - Подкатегории (Диваны, Кресла, Витрины и т.д.)
- **Product** - Товары с характеристиками, ценами и изображениями
- **ProductImage** - Дополнительные изображения товаров
- **ProductStock** - Управление остатками на складах
- **ProductRating** - Рейтинги товаров
- **Store** - Магазины/склады

## Функциональность

### 1. Каталог (`/catalog/`)
- Карточки всех категорий с изображениями
- Анимированные подменю при наведении на категории
- Плавные переходы и эффекты

### 2. Страница категории (`/catalog/<slug>/`)
- Список товаров категории
- Сортировка: по популярности, цене, новизне
- Карточки товаров с основной информацией

### 3. Страница подкатегории (`/catalog/<category>/<subcategory>/`)
- Товары конкретной подкатегории
- Хлебные крошки для навигации

### 4. Страница товара (`/product/<slug>/`)
- Детальная информация о товаре
- Галерея изображений с переключением
- Добавление в корзину
- Похожие товары

### 5. Корзина (`/cart/`)
- Список добавленных товаров
- Изменение количества
- Расчет итоговой суммы
- Оформление заказа

### 6. Админ-панель (`/admin/`)
- Управление категориями и товарами (требуется создать суперпользователя)
- Загрузка изображений
- Управление складами и остатками
- Просмотр рейтингов

**Примечание**: Система регистрации пользователей отсутствует. Сайт работает как публичный каталог, а управление контентом осуществляется через админ-панель Django.

## Дизайн и стилизация

### Цветовая схема:
- Основной цвет: `#444` (серый)

## Подготовка к продакшену

### Быстрый старт

Проект полностью настроен для продакшен развертывания. См. подробную инструкцию в [DEPLOY.md](DEPLOY.md)

### 1. Настройте переменные окружения

Скопируйте `.env.example` в `.env` и заполните значения:

```bash
cp .env.example .env
nano .env
```

Основные переменные:
```env
DEBUG=False
SECRET_KEY='сгенерируйте-уникальный-ключ'
ALLOWED_HOSTS=yourdomain.com,www.yourdomain.com

# Database (PostgreSQL рекомендуется)
DB_ENGINE=django.db.backends.postgresql
DB_NAME=mebel_db
DB_USER=mebel_user
DB_PASSWORD=strong_password

# Email
EMAIL_HOST=smtp.gmail.com
EMAIL_HOST_USER=your-email@gmail.com
EMAIL_HOST_PASSWORD=app-password
```

### 2. Технологический стек для продакшена

- **Веб-сервер**: Nginx (конфиг в `deployment/nginx.conf`)
- **WSGI-сервер**: Gunicorn (конфиг в `gunicorn_config.py`)
- **Process Manager**: systemd (сервисы в `deployment/`)
- **База данных**: PostgreSQL 14+
- **SSL**: Let's Encrypt (certbot)
- **Static Files**: WhiteNoise

### 3. Файлы конфигурации

Все готовые конфигурации находятся в директории `deployment/`:
- `nginx.conf` - конфигурация Nginx с SSL
- `mebel.service` - systemd сервис для Gunicorn
- `mebel-cron.service` / `mebel-cron.timer` - автоматические задачи

### 4. Безопасность

Проект включает:
- HTTPS redirect
- HSTS headers
- CSRF protection
- XSS protection
- Secure cookies
- Content Security Policy

### 5. Checklist перед деплоем

См. [PRODUCTION_CHECKLIST.md](PRODUCTION_CHECKLIST.md) для полного списка проверок перед развертыванием.

### 6. Мониторинг

Логи находятся в:
- `/var/www/mebel/logs/gunicorn_*.log` - логи Gunicorn
- `/var/www/mebel/logs/django_errors.log` - логи Django
- `/var/log/nginx/mebel_*.log` - логи Nginx
