# Cascate Porte - Мебельный каталог

Django-приложение для каталога мебели с функциональностью, аналогичной сайту cascate.ru.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Примените миграции:
```bash
python manage.py makemigrations
python manage.py migrate
```

3. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

4. Запустите сервер разработки:
```bash
python manage.py runserver
```

5. Откройте админ-панель:
```
http://127.0.0.1:8000/admin/
```

## Структура базы данных

### Модели:

- **Category** - Главные категории (Мягкая мебель, Столовые, Светильники и т.д.)
- **SubCategory** - Подкатегории (Диваны, Кресла и т.д.)
- **Product** - Товары (мебель)
- **ProductImage** - Изображения товаров
- **FilterCategory** - Категории фильтров (Популярные, По стилю и т.д.)
- **FilterValue** - Значения фильтров

## Функциональность

1. **Главная страница каталога** (`/catalog/`)
   - Отображение всех категорий с карточками
   - При наведении появляются подкатегории и информация
   - Анимации и плавные переходы

2. **Страница категории** (`/catalog/<slug>/`)
   - Список товаров в категории
   - Фильтры по различным параметрам
   - Сортировка товаров
   - Поиск

3. **Страница товара** (`/product/<slug>/`)
   - Детальная информация о товаре
   - Галерея изображений с переключением
   - Похожие товары

## Добавление данных

Через админ-панель Django можно:
1. Создать категории и подкатегории
2. Добавить товары с изображениями
3. Настроить фильтры
4. Управлять наличием товаров

## Особенности

- Адаптивный дизайн
- Плавные анимации при наведении
- Переключение изображений товаров
- Фильтрация и сортировка товаров
- Современный UI/UX
